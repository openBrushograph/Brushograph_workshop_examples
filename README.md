# Brushograph Workshop Examples

This folder contains example files for workshops using the **openBrushograph**.

## Purpose

- Provide ready-made example drawings and test patterns
- Offer calibration and alignment helpers
- Give participants demo files to quickly try out the openBrushograph

## Suggested structure

- `examples/`  
  Ready-to-use designs for demonstrations and exercises.

- `templates/`  
  Simple starting patterns that participants can remix during the workshop.

## Using these files with openBrushograph

1. Open an example or template in your preferred drawing or CAD software.
2. Adjust the design if needed (size, layout, lines, etc.).
3. Export in the format used in the workshop workflow.
4. Load it onto the openBrushograph following the workshop instructions.

Feel free to duplicate this folder, add your own examples, and share improvements back to the community.
